# تتراشاپ - واسط کاربری

این پنل مدیریت پیشرفته برای سیستم تتراشاپ می‌باشد.

## ویژگی‌ها
- مدیریت صفحات و محتوا
- جستجوی پیشرفته
- تنظیمات الگوریتم‌ها
- تحلیل‌های عملکردی

## راه‌اندازی

### روش ۱: با Python
\`\`\`bash
python3 -m http.server 8000
\`\`\`

### روش ۲: با Node.js
\`\`\`bash
npm install
npm run serve
\`\`\`

### روش ۳: با PHP
\`\`\`bash
php -S localhost:8000
\`\`\`
